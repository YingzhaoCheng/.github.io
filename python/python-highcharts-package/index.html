python-highcharts package

Posted on 2018-09-02 | In python | Visitors:
python package intrduction

Package intrduction

python-highcharts is a simple translation layer between Python and Javascript for Highcharts projects (highcharts, highmaps, and highstocks).
so 我们可以用python来画highcharts各种好看的图，而不需要去自己编写Javascript

Installation

python-highcharts supports Python 2.7 or 3.4+

注：还有其他库charts、pandas-highcharts等也可以画highcharts图，有兴趣的可以自行去了解一下
