什么是函数

  • 函数存储着一系列sql语句，调用函数就是一次性执行这些语句【降重】 
  • 但注意的是函数注重返回值，不注重执行过程，所以一些语句无法执行 
  • 函数并不是单纯的sql语句集合 
  • 函数只会返回一个值，不允许返回一个结果集【返回唯一】
